CentOS 7 安装smplayer


[TOC]

由于CentOS主要面向服务器用户，因此在个人软件方面比较贫乏
SMPlayer提供源码安装，但是其安装过程及其复杂

一、安装软件源：

以root权限或者sudo执行

1、安装依赖库

Nux Dextop库依赖于EPEL库，所有要先安装EPEL库(需要管理员权限)。如果安装过则跳过。

yum -y install epel-release

2、安装软件源

yum install http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-5.el7.nux.noarch.rpm

软件源可能更新，可在此处获取最新版本：http://li.nux.ro/download/nux/dextop/el7/x86_64/
查找：nux-dextop-release-0-5.el7.nux.noarch.rpm类似的源

二、安装smplayer：

yum install smplayer
