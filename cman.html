1、下载

wget https://src.fedoraproject.org/repo/pkgs/man-pages-zh-CN/manpages-zh-1.5.2.tar.bz2/cab232c7bb49b214c2f7ee44f7f35900/manpages-zh-1.5.2.tar.bz2

链接可能失效，可以在以下链接中获取最新中文包
https://src.fedoraproject.org/repo/pkgs/man-pages-zh-CN/

2、解压

tar -xjf  manpages-zh-1.5.2.tar.bz2

3、安装

切换工作目录

cd  manpages-zh-1.5.2

以下命令需在root账号下运行

安装

./configure --prefix=/usr/local/zhman --disable-zhtw

禁用zhtw，繁体中文

make

make install

4、设置别名

切换工作目录

cd ～

编辑配置文件

vim .bashrc

将下面这行添加到文件后面

alias cman='man -M /usr/local/zhman/share/man/zh_CN'

保存退出

重载参数

source .bashrc
